This project aims to create a mobile price classification system using a dataset from Kaggle. The dataset includes features related to mobile phones, and the goal is to classify them into four price categories: 0 (low cost), 1 (medium cost), 2 (high cost), and 3 (very high cost). The project will employ three key algorithms for classification:

  • Logistic Regression: A simple yet effective linear model for binary and multiclass classification tasks.
  • Random Forest Classifier: A robust ensemble learning method that combines multiple decision trees for improved accuracy.
  • Hybrid Classifier: This classifier may involve a combination of different algorithms or techniques, possibly including feature engineering, to enhance the classification performance.
    • The project's backend will be implemented in Python, utilizing libraries like scikit-learn for machine learning. The frontend will be developed using HTML, CSS, and JavaScript to create a user-friendly interface for inputting mobile features and obtaining price category predictions. The expected outcome is a user-friendly system that can accurately classify mobile prices based on the provided features.
